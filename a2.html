HTML Source (Google Chrome)

  1. Open your browser and navigate to the page (ESPN) that which you wish to view the HTML
  2. Right-click on the page to open the right-click menu after the page finishes loading.
  3. Click the menu item that allows you to view the source. In Internet Explorere, the menu item is “View Page Source.”
  4. When the source page opens, you'll see the HTML code for the full page. Scan through it to locate code for specific features and objects on the page, as desired.
  5. This is what we call "Crumbling the cookie".
